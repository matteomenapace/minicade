Sample Games

Here are some sample games to get you started.

  • Flyswat is a very simple game with art by @varelidi and code by @toolness.
  • Poundsplat, by @stenington, is a tiny side-scroller that uses the Coquette library.
  • Type-The-Word is a rather self-descriptive game by @epchristensen.

We're always looking for new games to add to this list. If you make one, just tweet a link to @toolness and he'll add it. Alternatively, you can edit this file and submit a pull request.
